research

monopoly



The script monopoly.py can be found in the misc repository mentioned in my "code" section, but it's my favourite script in that repository, so it gets its own section. It essentially simulates a very rudimentary Monopoly game. The positions and colours of all properties in the game, as well as railroads and utilities, are specified. The positions of Chance squares and Community Chest squares are also specified, and decks of these cards are generated and drawn from if the appropriate square is landed on.

A certain number of games are played with a certain number of dice rolls each (I haven't been able to find the average length of a Monopoly game in dice rolls), which generates data about the frequency with which each colour of property is landed on. The position on the board is affected by dice rolls, the "Go to Jail" square on the board, and certain Chance and Community Chest cards. The script can optionally generate plots of this data, and a simple printed version of a Monopoly board with each individual square containing the number of times they were landed on. Sample output for the code can be found above.
